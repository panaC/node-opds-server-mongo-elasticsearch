
# opds-server

OPDS2 server

NOT MAINTAINED

extracted from https://github.com/panaC/BookAssistant repo

## Stack

- node12
- typescript3
- nestjs
- mongoose
- elasticsearch

see src/constants.ts

## Description

Built with [Nest](https://github.com/nestjs/nest) framework TypeScript

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
